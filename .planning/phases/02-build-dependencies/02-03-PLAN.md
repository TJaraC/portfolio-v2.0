<objective>
Update `package.json` to: (1) change the build script from `tsc --noCheck` to `tsc --noEmit` enabling type-safe production builds (BUILD-01), and (2) remove the unused `recharts` and `prop-types` dependencies to reduce install surface area (BUILD-02, BUILD-03).

This plan depends on Plan 01 (02-01-PLAN.md) because `tsc --noEmit` will fail if the TypeScript errors from Plan 01 are not yet fixed.

Purpose: The `--noCheck` flag silently skips all TypeScript validation in production builds — a type error can ship undetected. Removing ~500KB of unused dependency surface area also reduces attack surface and install time.
Output: `package.json` with corrected build script and two fewer dependencies; clean `npm install`.
</objective>
