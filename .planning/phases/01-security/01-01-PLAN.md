<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Vercel CDN | HTTP response headers must be sent by the CDN before page load |
| Opened tab → Opener tab | window.open without noopener gives new tab a reference to window.opener |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | vercel.json (missing X-Frame-Options) | mitigate | Add `X-Frame-Options: DENY` in vercel.json headers array — prevents the site from being embedded in a frame by a malicious domain for clickjacking |
| T-01-02 | Tampering | vercel.json (missing X-Content-Type-Options) | mitigate | Add `X-Content-Type-Options: nosniff` — prevents MIME-type sniffing attacks where browser interprets a text/plain response as executable script |
| T-01-03 | Information Disclosure | vercel.json (missing Referrer-Policy) | mitigate | Add `Referrer-Policy: strict-origin-when-cross-origin` — limits referrer header exposure to cross-origin navigations |
| T-01-04 | Elevation of Privilege | window.open without noopener (Header.tsx ×4, Contact.tsx ×2) | mitigate | Pass `'noopener,noreferrer'` as third argument — prevents the opened tab from accessing window.opener and reassigning the parent page's location |
| T-01-05 | Denial of Service | vercel.json (missing Permissions-Policy) | mitigate | Add `Permissions-Policy: camera=(), microphone=(), geolocation=()` — restricts browser feature access; prevents a compromised third-party script from requesting sensor access |
| T-01-06 | Spoofing | vercel.json (missing HSTS) | mitigate | Add `Strict-Transport-Security: max-age=63072000; includeSubDomains; preload` — forces HTTPS, prevents protocol downgrade attacks |
| T-01-07 | Cross-Site Scripting | Content-Security-Policy with unsafe-inline | accept | Portfolio SPA uses GSAP inline styles extensively; restricting to nonce-based CSP requires a Vite plugin refactor out of scope. Risk is low: site has no user-generated content and no eval() usage |
</threat_model>
